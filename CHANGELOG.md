# 更新记录

## v2026.06.09 - 2026-06-09

### 主要更新

- 新增 TurnContext 结构化上下文，拆分客户文本、商品卡、订单卡、媒体消息和场景提示。
- 修复商品卡、订单卡元数据污染客户问题的问题。
- 支持价格分转元，例如 1161 解析为 11.61。
- 修复 CustomerAgent 中 `get_config` 未导入导致回复失败的问题。
- 售后场景收到图片或视频时，直接转人工处理。
- 售后场景反馈没有充电口、找不到插电口、在哪里插电时，直接转人工处理。
- 多订单待确认场景默认按售后处理，避免有订单客户被误判为售前。
- 移除图片单独消息在 AIReplyHandler 中提前回复的逻辑，让图片消息进入 CustomerAgent 统一处理。

### 注意事项

- 本版本不包含 runtime 分析数据、聊天记录、日志、cookie、token 或本地缓存文件。
- 更新线上服务后需要重启进程，Python 代码改动才会生效。
